Description Wil Clouser [:clouserw] 2010-06-23 23:21:13 PDT 
One of the new links on the Add-on details page is "Add to Collection."  See
location and layout on these mockups:
http://people.mozilla.com/~chowse/drop/amo/electric-bandwagon/v1/Addon_Details.png
http://people.mozilla.com/~chowse/drop/amo/electric-bandwagon/v1/Persona_Details.png
http://people.mozilla.com/~chowse/drop/amo/electric-bandwagon/v1/Interactions_Directories.png

And see the interaction flow here:
http://people.mozilla.com/~chowse/drop/amo/electric-bandwagon/v1/Interactions.png

This bug is for the full implementation of this widget on detail pages and on
listing pages.  Remember scalability was an issue when dealing with this widget
in remora.  Don't forget unit tests for things like "This add-on is already in
the collection, show it as checked."  See the spec for details, section "Add-on
Details page":

https://docs.google.com/Doc?docid=0Acwo2Bn17-PrZGZudHRobnJfNjVjYnFqMm5kZw&hl=en
Comment 1 Dave Dash [:davedash] 2010-07-06 12:19:14 PDT
